LLMPhysics Bot — Privacy Policy

Last updated: June 2026

This policy explains what data LLMPhysics Bot collects, how it is used, and how it is stored.

1. Data collected

The Bot processes the following data to perform moderation tasks:

  • Reddit user IDs — to enforce per-user posting quotas and track self-response violations
  • Post and comment IDs — to associate moderation actions with content
  • Post and comment text — to evaluate length limits and depth; for AI review, to send to Google Gemini
  • Post flair — to apply flair-specific rules
  • Moderator actions — to track comment removals for reporting purposes

The Bot does not collect email addresses, IP addresses, or any off-Reddit personal information.

2. Data storage

Devvit Redis — Quota counters, session state, and moderation flags are stored in Devvit's per-app Redis store. This data is scoped to the Bot's app installation and is subject to Devvit's data retention policies. Quota data expires automatically (rolling 24-hour window).

Supabase — Structured logs are written to a Supabase database operated by the moderator team. This includes: module name, event type, action taken, Reddit user ID, and post/comment ID. These logs are used for debugging and moderation oversight. No post or comment body text is stored in Supabase.

PDF review jobs — When a review is requested for a post with a linked PDF, the PDF URL and processing status are stored in Supabase until the review is complete.

3. Third-party services

Google Gemini API — When the Adversarial Reviewer is used, post content and linked material are sent to Google's Gemini API for analysis. Google's Privacy Policy applies to that processing.

Wikipedia / Zenodo — The Bot may fetch content from Wikipedia and Zenodo to support the !define command and PDF review. No user data is sent to these services.

4. Data sharing

Data collected by the Bot is not sold or shared with third parties except as described above (Google Gemini for AI review).

5. Data retention

Redis data expires automatically per the Bot's configured TTLs. Supabase logs are retained at moderator discretion. You may request deletion of log entries referencing your user ID by contacting the moderators via modmail.

6. Your rights

Reddit's own Privacy Policy governs Reddit's handling of your data. For data stored by the Bot, contact the r/llmphysics moderators via modmail.
