LLMPhysics Bingo — Privacy Policy

Last updated: June 2026

This policy explains what data LLMPhysics Bingo collects, how it is used, and how it is stored.

1. Data collected

The Game collects and processes the following data:

  • Reddit user IDs — to generate and store a bingo card per player per game post, and to track which tiles each player has marked
  • Reddit post IDs and comment IDs — to associate game events with subreddit activity
  • Post and comment text — to evaluate whether tile conditions have been met, including sending content to Google's Gemini API for AI-assisted validation
  • Post flair — required for certain tile conditions (e.g. "Missing the Joke" requires the Humorous flair)

The Game does not collect email addresses, IP addresses, or any off-Reddit personal information.

2. Data storage

Devvit Redis — Bingo card state (tile layout and marked status) is stored per user per game post in Devvit's per-app Redis store. This data is scoped to the Game's app installation. Game data expires automatically after 8 days of game post inactivity.

Event queue — Subreddit events (posts and comments) are queued in Redis for batch processing by the tile validator. Queued events expire after 48 hours.

3. Third-party services

Google Gemini API — Post and comment text is sent to Google's Gemini API to evaluate tile conditions that require contextual judgment (for example, determining whether a comment is earnestly criticising a humorous post). Google's Privacy Policy applies to this processing. Only content that is already publicly visible on r/llmphysics is sent.

4. Data sharing

Data collected by the Game is not sold or shared with third parties except as described above (Google Gemini for tile validation).

5. Data retention

Game state in Redis expires automatically after 8 days. You may request deletion of data associated with your user ID by contacting the moderators via modmail.

6. Your rights

Reddit's own Privacy Policy governs Reddit's handling of your data. For data stored by the Game, contact the r/llmphysics moderators via modmail.
